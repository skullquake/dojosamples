Dojo Query Module Demo

Example 0: Basic Querying

You can use dojo.byId to get elements by id attribute, assuming the id values are all unique. In that case you have to come up with unique id values, either programatically via the server, or in the client with JavaScript, or manually. The dojo/query module provides more general querying capabilities using CSS3 selectors.
