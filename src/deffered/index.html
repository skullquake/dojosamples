Augmenting Objects with Dojo

Example 0: lang.mixin

lang.mixing allows you to combine properties of objects. Note that a shallow copy is utilized.

Object A
Object B
lang.mixin(A,B)

You can mix multiple objects at the same time, e.g. lang.mixin(A,B,C,...N), and the properties will be mixed into the first argument.

Example 1: declare.safeMixin

declare.safeMixin operates on objects instantiated from classes created with dojo/_base/declare calls. declare.safeMixin does much the same as lang.mixin with three differences: 1) It can only mix one object at a time 2) it will not mix in constructors 3) function annotations will be added enabling this.inherited to work properly.

declared constructors expose an extend method, which uses declare.safeMixin under the hood.

Example 2: lang.extend

lang.extend mixes in on the prototype level, meaning that after applying it, all instantiated objects having the prototype to which extend was applied, will be affected immediately.
